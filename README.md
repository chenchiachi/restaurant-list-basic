# 餐廳清單

## 基本功能
- 使用者可以在首頁看到所有餐廳與它們的簡單資料：
  - 餐廳照片
  - 餐廳名稱
  - 餐廳分類
  - 餐廳評分

- 使用者可以再點進去看餐廳的詳細資訊：
  - 類別
  - 地址
  - 電話
  - 描述
  - 圖片
  
- 使用者可以透過搜尋餐廳名稱來找到特定的餐廳

## 安裝
- git clone  此專案
```
https://github.com/chenchiachi/restaurant-list.git
```
- cd 至專案資料夾
- 安裝npm 套件
- 使用npm開啟程式
